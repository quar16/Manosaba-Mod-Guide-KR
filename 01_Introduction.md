# 1. 소개

## 시나리오를 쓰고 싶은 분들께

### 이 Mod 프레임워크에 대하여
《마법소녀 마녀재판》은 Unity+Naninovel 엔진으로 개발된 AVG(어드벤처) 게임이며, 이 Mod 프레임워크는 BepInEx를 기반으로 Unity 엔진에 코드를 주입하여 `.nani` 시나리오 파일을 Naninovel 엔진에 로드하는 방식으로 작동합니다.
따라서 본 Mod 프레임워크가 로드하는 `.nani` 스크립트는 Naninovel 공식 표준을 완전히 따르며, 사용하는 명령어는 공식 지원 명령어와 게임 개발사가 자체 개발한 명령어의 집합체입니다.

어느 정도 영어 독해 능력이 있다면 Naninovel 공식 가이드를 읽어 공식적으로 제공되는 기초 엔진 기능을 빠르게 습득할 수 있으며, 게임 개발사가 자체 개발한 명령어는 게임 내 기존 스크립트를 통해 기능을 역으로 추론해야 합니다.

### 시나리오를 쓰려면 프로그래밍 기초가 필요한가요?
프로그래밍 기초는 전혀 필요하지 않습니다. Naninovel 공식 측에서 시나리오 작성과 관련해 매우 친절한 사용자 경험(UX) 설계를 해두었기 때문에, 코드 로직과 관련된 작업은 거의 발생하지 않습니다. 대화창 조작, 인물 표정, 배경 변경 등에 사용되는 몇 가지 명령어만 기억하면 됩니다.

### 왜 언팩(추출) 리소스와 게임 자체 시나리오를 직접 제공하지 않나요?
**《마법소녀 마녀재판》 방송 및 2차 창작 가이드라인**에 따라, 2차 창작 범위를 벗어나 언팩 리소스나 텍스트 스토리를 사용할 수 없으므로, 완성된 리소스 파일은 일절 제공하지 않습니다.

### 제가 만든 시나리오를 배포하고 싶은데, 주의할 점이 있나요?
본 Mod는 별도의 요구 사항이 없으며, 공식 **《마법소녀 마녀재판》 방송 및 2차 창작 가이드라인**만 준수하면 됩니다.
괜찮으시다면 소개란 같은 곳에 원작자(SherryAppleJuice)를 언급해 주시면 감사하겠습니다.
