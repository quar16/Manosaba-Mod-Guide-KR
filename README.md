# 🪄 마법소녀 마녀재판 (Manosaba) Mod 제작 가이드 (한국어)

이 저장소는 게임 **《마법소녀 마녀재판 (Mahou Shoujo no Majo Saiban)》**의 모드 제작 프레임워크 사용법을 한국어로 번역하여 정리한 문서입니다.

이 가이드는 중국어 원문 가이드를 기반으로 하며, 한국 유저들이 쉽게 시나리오를 창작하고 모드를 즐길 수 있도록 돕기 위해 작성되었습니다.

---
